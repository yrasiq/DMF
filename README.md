# DMF
Бот для WoW classic warlock. Забытый город - восток.

Бот предназначен для личного использования и имеет настройки, специфичные для конкретного ПК, персонажа и интерфейса. В таком виде вы не сможете его использовать. Но можете просто ознакомится с кодом проекта. Это законченная программа и ее дальнейшего изменения/улучшения не будет. Так же это мой первый проект на python и код во всех смыслах оставляет желать лучшего, но, тем не менее, все стабильно работает с середины 2020 г.
